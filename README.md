# Airport Information Kiosk

A gesture-controlled Airport Information Kiosk built using Python, OpenCV, and MediaPipe.

## Features

* Hand gesture navigation
* Flight search interface
* Flight details page
* Seat selection system
* Boarding pass display
* Real-time webcam interaction
* Contactless user experience

## Technologies Used

* Python
* OpenCV
* MediaPipe
* Computer Vision

## How to Run

1. Install dependencies

```bash
pip install opencv-python mediapipe numpy
```

2. Run

```bash
python airport_kiosk_v2.py
```

## Project Demo

This project demonstrates a touchless airport kiosk interface where users can navigate screens using hand gestures detected through a webcam.
